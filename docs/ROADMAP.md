# EchoFree roadmap

## v0.1 — Current prototype

- Custom Debian netinst ISO build
- Automated base installation
- Docker and Linux Voice Assistant
- First-boot terminal wizard
- Home Assistant ESPHome discovery
- Basic status command

## v0.2 — Hardware validation

- Test Intel N100 Beelink
- Test Dell OptiPlex Micro
- Verify UEFI and legacy BIOS boot
- Verify Realtek/Intel Ethernet firmware
- Validate USB speakerphone models
- Add deterministic audio-device selection
- Add rollback to previous LVA image tag

## v0.3 — Browser setup

- Local setup web page
- Room naming
- Audio test and level meter
- Wake-word selection
- Reboot and update controls
- Home Assistant discovery status

## v0.4 — Appliance operations

- Signed update manifest
- Backup/export configuration
- Clone-room profile
- Health endpoint
- Home Assistant sensor entities
- Read-only recovery boot option

## v1.0

- Tested ISO release
- Secure credential setup
- Checksums and release signing
- Documentation for Beelink and Dell
- Safe disk-selection UI
- Upgrade and disaster-recovery procedure
